# DeepSeek 提示词要求

## 官网首页与文章模板

请生成面向“闪电加速器”的中文官方网站发布首页和文章详情页模板。目标是符合 SEO 与 GEO，适合搜索引擎摘要、AI 搜索引用和结构化问答抓取。

要求：

- 首页标题语义围绕“闪电加速器-官方网站发布首页”。
- 首页首屏必须直接呈现品牌名、官网入口、备用入口、下载入口、更新时间和访问提醒。
- 文章页必须适合教程类、故障排查类、备用入口类和下载说明类内容。
- 页面必须加载快：不用外部字体、不用外部 CSS/JS、不用复杂动画、不引入前端框架。
- HTML 结构语义化：h1、h2、h3 层级清晰，FAQ 用可抓取结构。
- 文案不能虚假承诺，不写“永久免费”“100%可用”“官方认证通过”等不可验证表述。
- 设计要唯一、有官方发布页质感，避免通用博客模板、AI 落地页模板、单一紫蓝渐变套路。
- 移动端优先，首屏信息清楚，入口易点击。
- 不在页面可见文案里出现 SEO、GEO、AI 生成、模板、占位符等实现词。

## 文章生成

文章关键词示例：

- 闪电加速器官网下载
- 闪电加速器打不开怎么办
- 闪电加速器备用入口
- 闪电加速器使用教程

写作要求：

- 文章开头 2-3 句话直接回答用户搜索意图。
- 必须包含：直接结论、官网入口说明、备用入口说明、使用步骤、打不开原因、检查清单、风险提醒、FAQ。
- 不要编造具体下载地址、激活码、价格、节点数量、永久有效、速度承诺、账号或 token。
- 如果没有明确官方链接，用“请以本站首页当前展示入口和页面更新时间为准”。
- FAQ 问题要自然覆盖长尾搜索词。
- description 控制在 80-140 字。
- title 自然包含关键词，不堆砌。
